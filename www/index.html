Index of Apps:

Accelerometer : This app displays, in real time, the Accelerometer values X, Y, Z.
The orientation of the image icon also changes with orientation of the phone.

geoLoc : This app displays the current location in latitude and longitude.

touch : This app detects different kinds of touches(single finger , double finger, point etc) on the screen.

shakeDetect : App to detect and alert when device is "shaken".
"Shake" is defined as an overall change in acceleration(3 axes) greater than a threshold.

shellCmds : To run and display output of some supported shell commands like top, ps etc.

showAppList : App displays in real time a list of currently running tasks/apps in the system.

showAppList_1 : App displays in real time a list of currently running tasks/apps in the system
Slightly different logic compared to showAppList

AppUsageGraph : This app detects different kinds of touches(single finger , double finger, point etc) on the screen

AppV1 : An app to track and collect phone usage data. This app collects data like accelerometer, geo location,
app usage, ringer settings , phone calls and messages. This occurs at regular time intervals.
This data is then sent to a remote server database at regular intervals provided there is some network connection available.

Note: All the above apps were developed on PhoneGap 2.7.0
